## 背景
- 東京駅に近い23区内の物件は便利だが、値段が高そうなので悩んでしいる
- 条件ごとの価格帯の相場を知ることで、納得いく中古マンションを探したい

## 分析の期待効果
- 希望条件のどの項目を、どの程度まで妥協すれば、価格がどれだけ下がるのかを把握し、それをもとに顧客が条件を見直せるようにする
- 市場を理解した上での納得感のある住まい探しをサポート

## 分析の実施事項
- 市場の中古マンション価格の相場を可視化する  
  - 市場の相場とトレンドを理解

- マンション価格に影響している変数とその度合いを重回帰分析を用いて明らかにする  
  - 「価格」と「条件」のトレードオフを明らかにする
 
## ターゲット層：  
共に東京駅に通勤するパワーカップル

#### バリバリ働く多忙カップル
- 希望条件：
  - 電車所要時間が**15分未満**
  - 最寄駅まで**5分以内**

#### 将来重視の子持ちカップル
- 希望条件：
  - 間取りが**4DK/LDK以上**
  - 築年数が**15年未満**


#### ハイブリッド勤務のインドアカップル
- 希望条件：
  - 間取りが**4DK/LDK以上**
  - 専有面積が**100m²以上**
